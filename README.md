### 3. Spot Trading (CLOB)

SHOKEN uses a **Central Limit Order Book (CLOB)** model for spot trading.

**Why CLOB?**
- Transparent price discovery
- Real bid / ask dynamics
- No curve-based pricing

Spot trading is available in **early-stage beta** and will continue to evolve.

---

### 4. Prediction Markets

Prediction markets allow users to trade on real-world outcomes using YES / NO positions.

**Characteristics:**
- Probability-based pricing
- Orderbook-style liquidity
- Clear resolution criteria

Prediction markets are treated as **markets**, not gambling interfaces.
