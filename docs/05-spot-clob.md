# Spot Trading (CLOB)

## Overview

Spot trading in SHOKEN is based on a **Central Limit Order Book (CLOB)** model.

## Why CLOB

- Explicit bid / ask pricing
- Transparent liquidity
- True price discovery

## Beta Status

- Limited trading pairs
- Early-stage matching logic
- UI subject to frequent changes

Spot trading is functional but not feature-complete.
