# Perpetual Futures Trading

## Overview

The Perpetual Futures module allows users to trade leveraged long and short positions without expiration.

## Core Features

- Long / Short positions
- Leverage configuration
- Real-time price charts
- Position-level PnL tracking
- Paper trading (beta)

## Trading Flow

1. Select a perpetual market
2. Choose Long or Short
3. Set position size and leverage
4. Submit the position
5. Monitor open positions
6. Close position manually

## Risk & Limitations

- Liquidation mechanics may change
- Advanced order types are not yet available
- Execution logic is under active iteration

Perp trading should be considered **experimental during beta**.
