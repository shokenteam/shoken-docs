# Prediction Markets

## Overview

Prediction markets allow users to trade on real-world outcomes using YES / NO positions.

Prices represent market-implied probabilities.

## Market Structure

- Binary outcomes (YES / NO)
- Prices between 0.00 and 1.00
- Orderbook-style liquidity where available

## Trading Flow

1. Select a market
2. Review description and resolution criteria
3. Choose YES or NO
4. Enter trade size
5. Submit trade
6. Track position until resolution

## Resolution

Markets resolve based on predefined criteria.
Winning positions settle at full value.
Losing positions settle at zero.
